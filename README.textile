h1. Welcome to TOGAF Designer by Obeo Network

TOGAF (The Open Group Architecture Framework) is an industry standard architecture framework used to govern an enterprise information architecture. It provides a set of tools to modeled the four levels:Business, Application, Data and Technology. It's standardized by the Open Group.

These plugins are released under the EPL Open-Souce License.

They are distributed a sample integration of TOGAF Architecture Content Framework in Obeo SmartEA (https://www.obeosmartea.com).

To build the module you'll need Maven3 (with Tycho plugin) and run @mvn clean verify@
